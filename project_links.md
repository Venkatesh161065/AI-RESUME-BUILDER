### Description
A full-stack AI-powered Resume Builder application. It allows users to log in securely with Firebase Google Auth, build resumes using a dynamic real-time split-screen form, and generate professional ATS-friendly bullet points via OpenAI integration. Includes a Razorpay gateway to upgrade to a premium account, and an HTML2PDF exporter to download the final creation.
